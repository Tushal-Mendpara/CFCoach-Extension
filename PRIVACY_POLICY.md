# CFCoach Extension Privacy Policy

Last updated: 2026-04-06

## Overview
CFCoach is a Chrome extension that helps Codeforces users with profile analysis, daily practice mixes, and contest generation.

## Data We Process
The extension sends the following data to the CFCoach backend:

- Codeforces handle(s) provided by the user.
- Request parameters used to generate analysis, daily mixes, and contests.
- Selected modes/tags for contest and practice generation.

The extension stores a handle locally in browser storage (`chrome.storage.local`) to improve UX.

## Where Data Is Sent
Extension requests are sent to:

- `https://cpcoach.onrender.com`

## What We Do Not Collect in the Extension
The extension does not directly collect:

- Passwords
- Payment data
- Personal documents

## Backend Processing
Requests are processed by the CFCoach backend to return recommendations and generated results. Backend logs may include technical request metadata for reliability and debugging.

## Data Sharing
We do not sell user data. Data is not shared with unrelated third parties for advertising.

## Data Retention
Local extension storage can be cleared by the user via Chrome extension storage/site data controls.

Backend retention depends on server logging policy. If you want deletion requests supported, contact the maintainer.

## Security
We use HTTPS endpoints for extension-to-backend communication.
