Profile Analysis

Personalized strengths and gaps
How CFCoach computes these percentages: Tag percentage is an estimated mastery score from your accepted rate, wrong attempts, and recency of submissions for that tag. Overall Strength is the weighted summary across unlocked tags, and Contest Ratio reflects how much real contest practice you do versus practice-only solving.

What the numbers represent

  • Rating: Current Codeforces rating from profile data.
  • Overall Strength: Weighted mastery across unlocked tags.
  • Contest Ratio: Balance of contest practice vs upsolving.
  • Practiced Topics: Number of tags with enough history.

How to use this tab

  • Start with Top Focus Areas to pick weak links for this week.
  • Use Pattern Insights to avoid repeating similar mistakes.

Daily Mix

Adaptive daily practice batches
How Daily Mix is generated: Daily Mix gives 4 batches per day, with 6 problems in each batch. Batches 1 and 2 are weak-topic focused, while batches 3 and 4 are random general-practice to preserve topic breadth. We never suggest already solved problems and avoid same-day repeats.

Contest Generator

Balanced mashup for a group
How Contest Generator works: This feature analyzes all handles you enter, combines their skill profiles, and suggests a balanced contest set that is fair for the whole group. Random Balanced mode mixes topic and difficulty coverage, while Tag Filtered mode restricts by selected tags. You must create the mashup contest yourself on Codeforces and add the suggested problems shown here.

Mode meanings

  • Random Balanced: Multi-topic set with spread difficulty.
  • Tag Filtered: Restrict generation to selected tags only.
  • Problem Count: Final size of suggested mashup set.
